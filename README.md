## UI/UX Project

LELANG INDONESIA

Proyek dari mata kuliah Enterprise Database yang berfokus pada perancangan desain aplikasi mobile berdasarkan website resmi Lelang Indonesia (lelang.go.id). Desain ini dibuat untuk meningkatkan pengalaman pengguna dengan tampilan yang lebih responsif dan mudah digunakan pada perangkat mobile.

BUDDY

Proyek dari mata kuliah Digital Entrepreneurship yang berfokus pada pengembangan ide bisnis berupa aplikasi digital untuk mendukung kesehatan hewan. Aplikasi ini dirancang untuk membantu pemilik hewan dalam memantau kondisi kesehatan hewan peliharaan serta mendapatkan informasi dan layanan terkait perawatan hewan.
